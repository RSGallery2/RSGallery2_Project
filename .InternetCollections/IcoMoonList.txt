Joomla Icons   
Joomla 3.x comes with Bootstrap Framework integration and Icomoon icons (in font format and can be called by CSS classes) to help ease developers' life. However, before this post, we did not find anywhere that lists the included icons for our developers' reference. We stated looking into the Joomla core and found the icomoon.css (delivered along with Joomla). We generated the list of available icons as below (with their class name for usage reference):
