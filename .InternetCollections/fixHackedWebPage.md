## some tasks
What to do on a hacked web site
* shut down the site with a htaccess, so you don't spread malware
* scan files for hacked code
* check the logs for the hack (see change date of hacked files to narrow it down)
* check if there are outdated joomla + extensions
* copy the database + files somewhere secure (local, don't run the code :-) )
* delete everything and recover a backup which is before the hack (=> that's why you should chekc the logs)
* update everything to the latest version (take the time to remove unused extensions)
* remove htaccess
